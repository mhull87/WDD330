Javascript: Novice to Ninja Ch 8 Forms

  • access form elements with document.forms[]; more examples in code
  • form.submit(); submits automatically, button with type of submit or image can submit a form too
  • form.resert(); or button with type reset
  • form.action = '/an/other.url' submits the form to a different url to be processed
  • add event listeners for focus and blur to set back text and take it away when they start typing
    placeholder attribute is better
  • Form Controls
    • input
      • text input fields
      • password input fields *see code hero
      • checkbox input fields *see code hero
      • radio button input fields *see code hero
      • hidden input fileds
      • file input fields
      • lots of others number tel color date email etc.
    • select drop-down list - use multiple attribute to select multiple items
    • textarea
    • button
    • menu - is not well supported, used to creat a dropdown menu
  • Form Validation
    • use both client side and server side Validation
    • required attribute *see code hero
    • add eventlistener directly to the input field to provide instant feedback *see code hero
    • disable the submit button if there are errors on the form *see code hero
