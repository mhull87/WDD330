JavaScript Novice to Ninja Ch 9 The Window Object

  • The Browser Object Model (BOM)

    • access the window object by using window. or the this keyword
    • window.alert('Hello); pop up dialog box
    • window.confirm('Do you wish to continue'); pop up box for conformation
    • window.prompt('Please enter your name.'); pop up box with an input field
    • these stop the program and can be an issue if something else needs to process at the same time or is waiting for a callback function

    • Browser Information

    • window.navigator.userAgent // returns the browser type ex. Safari version 10 on Mac OS
    • window.location.href //returns the URL of the current page
    • window.location.protocol //returns the protocol used ex. https
    • window.location.host //returns the domain of the current URL and the port number
    • window.location.hostname // returns the domain of the current URL
    • window.location.port //returns the port or an empty string if the port is not explicitly stated in the URL
    • window.location.pathname //returns the path that follows the domain
    • window.location.search //returns ? followed by the query string parameters
    • window.location.hash //returns # followed by the fragment identifier
    • window.location.origin //returns the protocol and domain where the current page originated from
    • window.location.reload(); //force a reload of the current page
    • window.location.assign(); //loads another resource from a URL provided as a parameter
    • window.location.replace(); //same as above but the current page will not be stored in the session history
    • window.location.toString(); //returns a string of the whole URL

    • The Browser History

    • window.history can be used to access information about any previously visited pages in the current browser session
    • window.history.length //how many pages before the current page
    • window.history.go(); //navigate to a specific page 0 being current positive ahead negative back
    • window.history.forward //adead one page
    • window.history.back //back on page

    • popup.close();
    • window.moveTo(); //x and y coordinates of the screen
    • window.resizeTo(); //two demensions

      • Cookies

    • small files that are saved locally o a user's computer
    • stores information for faster loading
    • limited to 4KB of data and 20 cookies per domain
    • create a cookie, assign it to JavaScript's 'cookie jar'
    • document.cookie = 'name=Melissa';
    • document.cookie: //to see the contents of the cookie jar
    • const cookies = document.cookie.split("; ");
      for(crumb of cookies) {
      const [key,value] = crumb.split("=");
      console.log(`The value of ${key} is ${value}`);
      }
      //The value of name is Melissa
    • cookies are session cookies by default
    • const expiryDate = new Date();
      const tomorrow = expirtDate.getTime() + 1000 * 60 * 60 * 24;
      expirtDate.setTime(tomorrow);
      document.cookie = `name=Batman; expires=${expirtDate.toUTCString()}`;
      document.cookie = 'name=Batman; max-age=86400'
      //86400 secs = 1 day
    • document.cookie = 'name=Batman; path=/' or document.cookie = 'name=Batman; domain=sitepoint.com'; // allows the root directory to have access
    • add secure for HTTPS only
    • to remove a cookie set it to expire at a time in the past
    • cookie libraries: Cookies.js or jsCookie
    • window.setInterval(); //repeatedly calls the function every give milliseconds
    • function chant() {console.log('Beetlejuice');}
      const summon = window.setInterval(chant, 1000); //6
