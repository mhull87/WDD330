JSNinja Ch 1 - Programing Basics

  • JavaScript is made to be backward compatible with all versions
  • New primitive data type: Symbol
  • var still works but only use let or const now
  • keep number of global variables to a minimum
    • assignment of variables inside a block without using let or const will make it available globally too
  • backslash is used to escape special characters
    1. Single quote marks \'
    2. Double quote marks \"
    3. End of line \n
    4. Carriage return \r
    5. Tab \t
    6. To type a backslash, use \\
  • .repeat(n) repeats string literals
    • you must use the Symbol() function
    • put a unique id into the parentheses to act as the string representation of the symbol
  • you can have 2 variables point to the same symbol using
    • const a = Symbol.for('shared symbol');
    • const b = Symbol.for('shared symbol');
  • Numbers
    • Hexadecimal (base 16)
      • starts with 0x
      • used for color codes
      • ex: 0xAF (A represents 10, F represents 15 = 175)
    • Octal literals (base 8)
      • starts with 0o
      • ex: 0o47 (4 eights and 7 units = 39)
    • Binary literals (base 2)
      • starts with 0b
      • ex: 0b1010 (1 eight, 0 fours, 1 two and 0 units = 10)
    • Exponential Notation
      • multiply by 10 to the power of
        • scientific notation or standard form
      • ex: 1e6 (1 multiplied by 10 to the power of 6 = 1000000)
      • ex: 2E3 (2 multiplied by 10^3 = 2000)
      • decimal values can be created by using a negative index value
        • ex: 2.5e-3 (2.5 multiplied by 10 to the power of -3 (0.001) or 0.0025)
    • .toExponential();
      • JavaScript will confuse a dot with a decimal point
        • fix this by:
        • use two dots (one is the decimal one is the dot notation
          • ex: 5..toExponential(); = 5e+0
        • put a space before the dot
          • ex: 5 .toExponential(); = 5e+0
        • always write integers as a decimal
          • ex: 5.0.toExpontntial(); = 5e+0
        • place the interger in parentheses
          • ex: (5).toExponential(); = 5e+0
        • assign the number to a constant
          • ex: const number = 5;
            number.toExponential(); = 5e+0
    • toFixed();
      • rounds a number to a fixed number of decimal places (returned as a string)
      • ex: const PI; = 3.1415926 = undefined
        PI.toFixed(3); = 3.142
    • toPrecision();
      • rounds a number to a fixed number of significant figures (returned as a string)
      • ex: 325678..toPrecision(2); = 3.3e+5
      • ex: 2.459.toPrecision(2); = 2.5
    • Incrementing
      • using an incrementing value before (++number) will return the original number and then add 1
        • ex: number++ will return 6, and then increment it to 7
      • after (number++) will return the number already incremented
    • Infinity
      • it will return infinity (or -infinity) if the numbr is too big for the program to handle
    • Bitwise (converts to bitwise (ones and zeros) and back)
      • ~ is the bitwise not (!) operator
      • & is the bitwise and (&&) operator
      • | is the bitwise or (||) operator
      • ^ is the bitwise or exclusive operator
      • << and>> are shift operators and will multiply or divide by 2
        • ex: 3 << 1; (multiply by 2=6)
        • ex: 16 >> 1; (divide by 2 = 8)
        • ex: 5 << 3; (multiply by 2 cubed=40)
    • do not use ^ for exponents. use **
    • for checking equality (== and ===) Nan wont work. use Number.isNaN() to check

    • JSNinja Ch 2 - Arrays

  • Arrays
    • delete arr[3];
      • delete array item at index 3 and leave it empty (undefined)
      • better to use .splice();
    • .length;
    • .length - 1;
    • .pop();
      • remove the last item
    • .shift();
      • remove the first item
    • .push('item');
      • adds new item to the end
    • .unshift('item');
      • adds new item to the beginning
    • .concat();
      • merges arrays
      • ex: arr2 = arr1.concat(['arr1item0', 'arr1item1', 'arr1item2'];
    • spread operator ...
      • spreads the items and adds them to a new array
      • ex: arr2 = [ ...arr2, ...['arr1item0', 'arr1item1', 'arr1item2'] ];
    • .join();
      • turns the arry into a string list seperated by commas
      • choose a seperator other than a comma by placing itinside the parentheses
        • ex: arr2.join(' & ');
    • .slice();
      • creates a temporary sub array but does not remove from the orrigional array
      • ex: arr1.slice(2,4);
        • starts at index 2 and ends on index 3
    • .splice()
      • removes items from the array and inserts new items
      • arr1.splice(3, 1, 'newitem');
        • starts at index 3 and replaces 1 item with newitem
        • .splice(); will return the removed item(s)
      • can be used to insert without removing
        • ex: arr1.splice(4, 0, 'item');
      • ex: arr1.splice(3,1); will delete and item without replacing it
    • .reverse();
    • .sort();
      • permanently changes array
      • string: alphabetical
      • numbers: by 1st digit
        • you can change this default by using a callback function for the argument
    • .indexOf();
    • .includes();
      • can add an extra parameter to indicate which index to start the search from
    • flatten multidementional arrays
      • const summer = ['Jun', 'Jul', 'Aug'];
        const winter = ['Dec', 'Jan', 'Feb'];
        const nested = [ summer, winter ];
        [ [ 'Jun', 'Jul', 'Aug' ], [ 'Dec', 'Jan', 'Feb' ] ]

        const flat = [...summer, ...winter];
        [ 'Jun', 'Jul', 'Aug', 'Dec', 'Jan', 'Feb' ]
    • use sets to remove any repeated items
      • new Set([1,2,3]);
      • .add();
      • .size();
      • .has();
      • .delete();
      • .clear();
        • clears the entire set
      • to an array
        • ex: arr1 = [...set1] or
        • ex: arr1 = Array.from(set1);
  • maps
    • used for storage and retrieval of key value pairs.
    • new Map();
    • .set();
    • .get();
    • .has();
    • .size();
    • .delete();
    • .clear();
    • Array.from(map1);
  • Ternary operators
    • condition ? code if true : code if false
  • for of loop
    • ex: for (const value of arr1) {
      return value;
      }
    • needs to be declared as a const
  • loop through objects, sets, and maps with key(), value(), or entries()

    • JSNinja Ch 3 - Functions

  • Functions
    • .name() can be used for debugging to see which function is causing the problem
    • multiple arguments in a function or unknown amount use:
      • function rest(...args) {
        return args;
        }
    • default parameters
      • assign it a value inside the function parameters
      • ex: function hello(name='World) {
        return `Hello ${name}!`;
        }
      • can use parameters and defaults in a function
    • Callback
      • you can pass a function as a parameter in another function. This is a callback.
      • function sing(song, callback) {
        callback();
        }
      • ex: function sing('Let It Go', dance);
      • dance does not need () because it is being called by its var
      • you can use sort() as a callback to properly sort numbers in an array by subtracting them
        • ex: function numerically(a,b) {
          return a-b;
          }

          [1,3,12,5,23,18,7].sort(numerically);
      • Array iterators (callbacks)
        • can be chained
        • forEach();
        • .map();
        • .reduce();
        • .filter();
        • .reduceRight();
        • .every();
        • .find();
        • .some();
