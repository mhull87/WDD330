JSNinja Ch 1 - Programing Basics

  • JavaScript is made to be backward compatible with all versions
  • New primitive data type: Symbol
  • var still works but only use let or const now
  • keep number of global variables to a minimum
    • assignment of variables inside a block without using let or const will make it available globally too
  • backslash is used to escape special characters
    1. Single quote marks \'
    2. Double quote marks \"
    3. End of line \n
    4. Carriage return \r
    5. Tab \t
    6. To type a backslash, use \\
  • .repeat(n) repeats string literals
    • you must use the Symbol() function
    • put a unique id into the parentheses to act as the string representation of the symbol
  • you can have 2 variables point to the same symbol using
    • const a = Symbol.for('shared symbol');
    • const b = Symbol.for('shared symbol');
